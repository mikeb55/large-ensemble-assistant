## Orchestration Choices by Movement

### I. Mingus Blues Cathedral (mm. 1-12)

**Opening Texture (m.1-3):**
- Cello establishes growling C2 pedal (sul ponticello)
- Double Bass supports with pizz → arco
- Guitar provides Cm7 harmonic pad (pp, arpeggiated)
- **m.3: Cello SOLO** performs Bass Poetry gesture (C3-Eb3-D3)
- Viola emerges on Ab3 color tone, preparing for melody

**Flugelhorn Entry (m.4-9):**
- Flugelhorn takes lead melody (syncopated entry beat 4)
- Long slur arc preserved across m.4-5
- Clarinet provides shadow line (3rds below, color)
- Viola and Cello build counterlines
- Guitar maintains harmonic rhythm with chord changes
- Double Bass provides root motion (half notes)

**Climax Build (m.10-12):**
- **Violin takes lead** with simplified line (Rhythmic Honesty)
- Flugelhorn moves to harmony
- Clarinet doubles violin 8va for brilliance
- Full string texture, staggered entries
- **m.12: Phrase Messiness** — Violin enters early (beat 6) with C5, anticipates F#5 on beat 7
- Cross-fade with Flugelhorn clearing out

### II. Gil's Orchestral Canvas (mm. 13-16)

**Gil Evans-Inspired Texture:**
- Violin continues from spillover, takes lead (*dolce, floating*)
- Guitar provides Dmaj9 shimmer (harmonics, let ring)
- Flute adds long tones on extensions (A, E)
- Viola offers gentle counterline (B3-D4 oscillation)
- Cello holds D2 pedal (con sordino)

**Fragility Moment (m.15):**
- **Violin SOLO** — A5 dotted half → G5 → F#5
- Marked *fragile, senza vibrato*, pp
- **ALL OTHERS TACET** (or single ppp harmonic glow)
- Guitar: one Gmaj7 chord, let decay
- Cello: sul tasto G2, barely audible
- Creates emotional center of the movement

**Recovery (m.16):**
- Violin continues G5 held, rubato
- Viola re-enters with counterline
- EbMaj7 color emerges (key change preparation)

## Wheeler/Schneider Aesthetic

The orchestration applies these hybrid chamber principles:

1. **Transparency** — Never more than 4-5 active voices in chamber passages
2. **Shared breathing** — Instruments phrase together as families
3. **Color rotation** — Same pitch passes between timbres (Klangfarben m.21)
4. **Dynamic matching** — No single instrument dominates the texture
5. **Modern harmony** — Extensions (9ths, 11ths) used freely in voicings
6. **Open spacing** — Avoids muddy middle register buildup
